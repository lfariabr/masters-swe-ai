# Cloud Computing Fundamentals - CCF501

## Subject Introduction
Cloud Computing Fundamentals provides integral knowledge of cloud service components and its current and future applications in industry automation. Students will explore the cloud service evolution and its services through software, platform, and infrastructure technologies. In addition, this subject provides in-depth knowledge about the different cloud deployment models, cloud segments, cloud security, and governance. Students will gain practical experience in designing and implementing cloud applications and other services on a variety of cloud computing platforms to meet stakeholder requirements.

## Subject Description
This subject provides a fundamental understanding of Cloud Computing. Students will be able to understand how the cloud computing infrastructure has evolved from the traditional IT infrastructure and what business advantages it brings. In addition, they will also learn the different Cloud Segments and Cloud Deployment models and the key players in the market. The subject also provides knowledge of Cloud Services and Cloud Security.
