### 1) In cloud adoption planning, what is the main problem with **over-provisioning** capacity?
A. It improves customer experience too quickly  
**B.** It wastes resources and increases inefficiency  
C. It prevents any future scaling  
D. It removes the need for governance  

### 2) Which NIST essential characteristic allows a customer to provision resources automatically without provider human interaction?
A. Broad network access  
B. Measured service  
**C.** On-demand self-service
D. Resource pooling  

### 3) For ABC Enterprise (start-up, fast growth, cost sensitivity), which cost model from the module content is the best fit?
A. Upfront perpetual hardware investment  
**B.** Pay-as-you-go usage model  
C. Fixed annual hardware replacement cycle  
D. Internal charge-back only, with no metering  

### 4) Which deployment model is defined as combining public and private environments while keeping each environment distinct?
A. Public cloud  
B. Private cloud  
C. Community cloud  
**D.** Hybrid cloud  

### 5) Which cloud service model best matches: "You manage applications and data, while the provider manages runtime, OS, and infrastructure"?
A. IaaS  
**B.** PaaS  
C. SaaS  
D. Community as a Service  

### 6) In the IaaS/PaaS/SaaS comparison, which model places the **most** management responsibility on the customer?
A. SaaS  
B. PaaS  
**C.** IaaS  
D. FaaS  

### 7) Which challenge is most closely associated with cloud **governance** in the module notes?
A. Inability to access the internet from any device  
**B.** Uncontrolled cloud usage and policy violations  
C. The absence of virtualisation in cloud systems  
D. No available deployment model choices  

### 8) A company must keep highly sensitive regulated data under strict control but still wants to scale variable workloads on demand. Which deployment strategy best fits?
A. Public cloud only  
B. Private cloud only  
**C.** Hybrid cloud  
D. Community cloud only  

### 9) In a VPC context, what is the most accurate statement from the module material?
A. A VPC is the same as a traditional on-prem private cloud  
**B.** A VPC is a logically isolated environment on shared public cloud infrastructure  
C. A VPC removes all need for access control policies

A. It requires higher upfront CapEx than on-prem systems  
B. It increases manual provisioning effort over time  
**C.** It enables elastic scaling and cost growth aligned with demand
