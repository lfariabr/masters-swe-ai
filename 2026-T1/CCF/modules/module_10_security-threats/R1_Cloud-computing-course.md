# Cloud Security
About this Course

Cloud computing is an emerging technology that allows storing, managing, and processing data remotely utilising servers in a network as opposed to the traditional way of using local servers. This technology has several benefits to individuals and businesses including, but not limited to, working remotely, automatic software update, minimum operational cost, flexibility, scalability, and sustainability. Because of the use of remote servers, there are a wide range of security concerns that otherwise would not exist in the conventional method of using local servers. 

Some of these security concerns include data breaches, insecure APIs, data loss, cloud services abuse, and malware injection, just to name a few. As we move in the era where cloud computing is becoming a part of everyday life, cloud security is a must for all of us. This short course shows you different areas of cloud security and will discuss elements of best practice. 

## Course Objectives
Recognise cloud computing characteristics and definitions.
Understand the main cloud services and deployment models.
Understand vulnerabilities and common security concerns.
Distinguish the different techniques, technologies and best practises in cloud security.
