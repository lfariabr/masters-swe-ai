# Dataset - JHU COVID-19 confirmed global time series

- File: `time_series_covid19_confirmed_global.csv`
- Source: Johns Hopkins University CSSE, via Humdata.org
- URL: https://github.com/CSSEGISandData/COVID-19 (csse_covid_19_data/csse_covid_19_time_series/)
- Coverage: cumulative confirmed cases per Country/Region (and Province/State), daily, 22 Jan 2020 to 9 Mar 2023 (the JHU series stopped updating 10 Mar 2023, so this is the final archived version).
- Shape: 289 location rows x 1143 date columns + Province/State, Country/Region, Lat, Long.
- Used as-is, no modifications (per the brief).

Citation: Humdata.org. (2020). Novel Coronavirus (COVID-19) cases data. https://data.humdata.org/dataset/novel-coronavirus-2019-ncov-cases
